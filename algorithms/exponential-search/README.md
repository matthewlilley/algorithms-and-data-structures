# Exponential Search

In computer science, an exponential search (also called doubling search or galloping search or Struzik search) is an algorithm, created by Jon Bentley and Andrew Chi-Chih Yao in 1976, for searching sorted, unbounded/infinite lists. [Wikipedia](https://en.wikipedia.org/wiki/Binary_search_algorithm)
